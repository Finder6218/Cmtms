# 化工物资运输管理系统

系统包含：
1、基本信息：司机信息、商品信息、客户信息、车辆信息、聘请车队

2、运输管理：运输登记、应付登记、应收登记

3、查询统计：按车号、客户查询，按客户查询，司机工资，按车号、客户名称支出查询，按客户名称支出查询，应收应付汇总，聘请车辆查询，聘请车辆应付查询，应收款客户提醒

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
